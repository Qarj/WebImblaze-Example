Welcome to the WebInject Example Pages.

If you want to get an idea what the WebInject Framework test run results look like, then you are in the right place.

Overall Summary.

Example WebInject Overall Summary Results

The Summary page shows overall statistics for all WebInject test run results for the day. This page shows one line for each batch of test run results. You can click on one of the lines to see what test case files made up that batch.

Batch Summary.

Example Batch Results

You would typically group up all the test case files that make up a complete "Regression" set into a single set of results. That way you can easily ensure that everything passed. In the example various kinds of failure states are shown to give you an idea how that looks. If you click on the run number (far left column), it will take you through to the individual run results.

Individiual Run Results.

Example Run Results

The individual run results show the status of each test step. If a test step fails, additional information is shown on why it failed. If you click on one of the step numbers, you will be taken through to the step result.

Step Result.

Example Step Result

The individual step result will render the page html for you so it is easy to what went wrong (if something did go wrong). You can also click to expand the Request and Response headers.
